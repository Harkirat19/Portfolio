About Me

I’m Harkirat Singh, a dedicated software developer currently pursuing 3rd year of my Bachelor of Technology (Honours) in Software Development at Seneca Polytechnic. With a strong foundation in both mobile and web app development, I thrive on crafting solutions that are efficient and scalable—because who doesn’t love a well-functioning app?

My technical skills cover a range of programming languages including Python, C, C++, HTML, CSS, JavaScript, TypeScript , and Node.js. Around the Globe
GitHub Repo Live Demo
Description

Around the Globe is an interactive website that uses the REST Countries API to provide detailed information about countries worldwide.

Key features include searching by name, filtering by region, viewing country details, and navigating neighboring countries.

The responsive design, along with light and dark mode toggles, ensures a seamless user experience on all devices.

  • REST Countries API
Space Fighter
GitHub Repo
Description

Space Fighter is a 2D game built with Python's Pygame library, where players control a spaceship to shoot down enemies moving toward it. The game ends if an enemy reaches the spaceship, displaying the score and credits.

Features include custom sprites, sound effects, background music, and intuitive controls for easy gameplay.

Tools & Technologies
  • Python
  • Pygame Library
  • Random Library (for enemy positioning)
  • Math Library (for collision detection)
  • Pygame.mixer (for music & sound effects)
